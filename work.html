My work

This was my first try ever, using HTML and CSS. I recreated a landing page for my Etsy Shop a MAR Berlin and I have to say, I was really proud of that first exercise! I learned that it can take hours to get minute design details right, but to me, that is deinitely part of the fun. I love design and trying out as much as getting the code to work.

For my second project, I coded a Weather App, using JavaScript, HTML and CSS as well as a weather API. JavaScript functions were definitely a step up from the basics, but the relief and joy was incredible, when I finally got things to work after hours of Google searches... It is incredible what you are able to do with these and I can not wait to learn more.
